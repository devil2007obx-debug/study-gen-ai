# 🚀 StudyGen AI

**StudyGen AI** is a Generative AI powered study assistant that converts notes into summaries, quizzes, flashcards and study plans automatically.

## 📝 Problem Statement
Students spend too much time reading notes but struggle to identify important topics, questions and revision material. Our AI Study Assistant automatically converts notes into summaries, quizzes and study plans to make studying faster and smarter.

## 💡 Solution & Features
Users simply paste their notes or upload study materials. AI automatically generates:
1. **Summary**: A concise overview of the notes.
2. **Important Questions**: 5-7 key concepts tested.
3. **MCQ Quiz**: Multiple choice questions to test knowledge.
4. **Flashcards**: Front/Back format for active recall.
5. **Topic Explainer**: Analogies for complex subjects.
6. **Study Timetable**: A 3-day practical study plan.

## 🛠️ Technology Used
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript
- **Backend**: Python, Flask
- **AI**: Google Gemini API (`gemini-1.5-flash`)

## 🏃‍♂️ How to Run the Demo
1. This project is **100% Serverless Frontend**! You don't need to install Python or a backend database to run it.
2. Open `script.js` in a text editor.
3. Find the line: `const GEMINI_API_KEY = 'AIzaSyA...';` and paste your valid Gemini API key there. *(The previous key was invalid).*
4. Open `index.html` in your web browser. 
5. Click **Get Started Free**, paste some notes into the input box, and click **Generate Planner** or any other tab!
